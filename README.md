# DesenvolvimentoSistemas-SENAI
Este repositório foi criado para organizar todas as atividades, exercícios, projetos e anotações desenvolvidos durante a disciplina de Desenvolvimento de Sistemas no SENAI. Aqui estarão registrados os conteúdos estudados ao longo do curso, desde os fundamentos da lógica de programação até conceitos mais avançados utilizando a linguagem Java.
